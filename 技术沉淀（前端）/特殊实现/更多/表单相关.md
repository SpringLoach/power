### 用户名脱敏

>两个字，则脱敏第二个字
>
>三个字，则脱敏第二个字
>
>三字以上，则脱敏第二和第三个字

```javascript
// 用户名脱敏
export const treatName = function (str) {
  if (str.length == 2) {
    return str.substring(0, 1) + '*'
  } else if (str.length == 3) {
    return str.substring(0, 1) + "*" + str.substring(2, 3)
  } else if (str.length > 3) {
    return str.substring(0, 1) + "*" + '*' + str.substring(3, str.length)
  }
  return ''
}
```
