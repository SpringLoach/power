## vue/h5生成二维码

```vue
<!-- 脚本 -->
<script setup lang="ts">
import QRCode from "qrcode";

const demo = ref('')
async function createQRCode() {
  demo.value = await QRCode.toDataURL('https://www.bilibili.com');
}()
</script>

<!-- 模板 -->
<img :src="demo" alt="">
```

:whale: 当 QRCode.toDataURL 接收的字符串为链接时，扫码将打开对应网址；否则将内容输出到页面。
