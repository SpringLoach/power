


### 重新触发对象的响应式

> 适用于给响应式的对象添加属性后，补充响应式的情景。

```javascript
//成功触发响应
this.activityInfo = Object.assign({}, this.activityInfo)

//失败触发响应
this.activityInfo = Object.assign(this.activityInfo， {})
```
