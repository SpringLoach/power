


#### 行内元素与块级元素  
> 空元素：在 HTML 中，通常在一个空元素上使用一个闭标签是无效的。例如 `<input type="text"></input>`
> 
> 行内元素：不可以设置宽高，不独占一行。
> 
> 块级元素：可以设置宽高，独占一行。


类型 | 栗子
:-: | :-:
行内元素 | a b span **img** input select strong
块元素 | div **ul** ol **li** dl dt dd h1-6 p
空元素 | `<br> <hr> <img> <input> <link> <meta> <embed>`

#### 注释  

类型 | 方式
:-: | :-:
HTML | `<!-- 在此处写注释 -->`
CSS | /\* 单行或多行注释 \*/
JavaScript | `// 单行` `/* 多行 */`
