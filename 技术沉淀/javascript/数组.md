## 数组

#### 数组拆分两半

> 源数组长度未知，拆分的第一个数组长度固定（有最大值）。

```javascript
let x = [3, 4, 5];
let y = x.splice(0, 10);
console.log(x);   // []
console.log(y);   // [3, 4, 5]
```
