### 点击标题跳转到对应元素

>  其实更好的做法是利用 `id` 指向唯一元素或用 `ref` 指向子组件。 

```javascript
// n 表示第几部分的内容
handleClickTheme(n) {
  let itemList = ['section1', 'section2' ,'section3']
  let item = itemList[n-1];
  document.getElementsByClassName(item)[0].scrollIntoView({behavior: "smooth"});
  // 该属性用于动态添加激活类
  this.activeItem = n;
}
```
